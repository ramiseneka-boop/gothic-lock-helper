Gothic Lock Helper — PWA

Как запустить:
1. Загрузите содержимое этой папки на Vercel, Netlify или GitHub Pages.
2. Откройте опубликованный HTTPS-адрес на iPhone.
3. Safari → Поделиться → На экран «Домой».

Важно:
- Через обычный локальный файл PWA и Service Worker не устанавливаются.
- Приложение поддерживает от 2 до 12 плиток.
- Количество отверстий фиксировано: 7.
