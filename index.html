🔓 Gothic Lock Helper

Работает с любым количеством плиток. Без таблицы Гаусса: ты только отмечаешь позиции и реакции.

1Настройка замка

Плитка №1 — первая в твоём порядке. Главное, чтобы ты использовал один и тот же порядок везде.

2Текущие позиции

Для каждой плитки выбери отверстие, где сейчас находится её штифт.

Целевое отверстие для всех штифтов
3Проверка связей
Для каждого теста сдвинь выбранную плитку ровно на один щелчок в рабочую сторону. Затем отметь, что сделали остальные плитки. Перед следующим тестом верни замок в исходное положение.
● не двигалась→ в ту же сторону← в противоположную
